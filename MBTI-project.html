MBTI Personality Classification Using Machine Learning

Python Excel
About the Project

This project applies machine learning to automate MBTI personality type classification using 60,000 structured survey responses across 60 numerically encoded behavioral questions. Five classification models were built, tuned through grid search hyperparameter optimization, and evaluated across accuracy, precision, recall, F1-score, and cross-validation to identify the most reliable classifier for 16-class personality prediction.

Problem Statement

Manually administering and scoring MBTI assessments is slow, subjective, and impractical at scale. Organizations in human resources, education, and behavioral research need faster, more reliable methods to derive personality insights from structured data. The goal was to build a multi-class machine learning framework capable of accurately classifying individuals into one of 16 MBTI personality types using survey data, while determining which algorithms generalize best and which questions carry the most predictive weight.

Output Interpretation
  • KNN achieved the highest overall accuracy at 98.77% with cross-validation score of 98.59% confirming strong generalization, while SVM demonstrated the best consistency with the tightest variance at 0.11%
  • Baseline stratified model scored only 6.22% accuracy, establishing a clear benchmark and validating the 16x performance improvement delivered by learned models
  • Logistic Regression achieved 91.82% accuracy while Random Forest and XGBoost both reached 97.63%, providing a comprehensive performance comparison across model families
  • Dichotomy level SVM binary classification across all four MBTI dimensions returned 99% precision, recall, and F1 score with accuracies ranging between 98.58% and 98.78%
  • Results confirm that machine learning can reliably automate MBTI classification with near perfect accuracy on structured survey data, with practical applicability in HR, team building, educational guidance, and behavioral research
